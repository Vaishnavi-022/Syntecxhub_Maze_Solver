# Maze Solver using A* Algorithm

## Project Description
This project implements a Maze Solver using the A* Search Algorithm in Python.

The algorithm finds the shortest path from Start (S) to Goal (G) while avoiding obstacles (walls).

## Features
- A* Search Algorithm
- Manhattan Distance Heuristic
- Shortest Path Finding
- Maze Visualization using Matplotlib

## Technologies Used
- Python
- NumPy
- Matplotlib

## Output
The program displays the maze and highlights the shortest path from Start to Goal.
