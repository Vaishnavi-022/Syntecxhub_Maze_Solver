# Syntecxhub Maze Solver 🧩

## 📌 Project Overview
This project is a Maze Solver built using the A* (A-Star) Search Algorithm in Python. 
The main objective of this project is to find the shortest possible path from the start point (S) to the goal point (G) while avoiding obstacles (walls) in the maze.

The A* algorithm is widely used in Artificial Intelligence for pathfinding and graph traversal problems because of its efficiency and accuracy.

---

## 🎯 Objective
The objective of this project is to understand and implement the A* Search Algorithm for solving maze problems and finding the shortest path efficiently.  
This project helps in learning important AI concepts such as:
- Pathfinding Algorithms  
- Heuristic Functions  
- Search Optimization  

---

## ⚙️ Features
- Finds the shortest path from Start (S) to Goal (G)
- Avoids obstacles (#) in the maze
- Uses Manhattan Distance as the heuristic function
- Displays the shortest path clearly
- Beginner-friendly implementation in Python

---

## 🧠 Algorithm Used
This project uses the **A* (A-Star) Search Algorithm**, which combines:
- Cost from start node (g)
- Heuristic estimate to goal (h)
- Total cost function (f = g + h)

The heuristic used in this project is **Manhattan Distance**, which improves search efficiency.
